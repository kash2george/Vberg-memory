# VBerg

A visual way to remember words and sentences.

---

## ✨ Idea

VBerg transforms words into visual structures.

Instead of memorizing directly, you remember:
- position
- grouping
- visual relationships

Inspired by real learning struggles like dyslexia.

---

## 🎮 What you can do

- Type a sentence → words are split into groups
- Type a single word → split into visual chunks
- Drag words between groups
- Resize groups with scroll
- Move groups horizontally

---

## 🧠 Why it works

People don’t remember things they see directly.
They remember context, position, and structure.

VBerg uses that.
